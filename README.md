# React Shopping Cart


**Shopping Cart Project** built with **React JS**, using **Context API** & **useReducer Hook**.

<br/>

## Features:

- #### Add to Cart
- #### Remove from Cart
- #### Calculates Total
- #### Resonsive 

<br/>

## Technologies used:

- #### **React JS**
- #### **SASS**



<br/>

## Setuo Scripts:

#### `npm install`

#### `npm run`
